Initializes the current module with the given name.

Subclasses should call this constructor before creating other modules or variables such that those modules are named correctly.

Args:
name: An optional string name for the class. Must be a valid Python

identifier. If name is not provided then the class name for the current instance is converted to lower_snake_case and used instead.
